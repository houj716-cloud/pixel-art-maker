# 像素画生成器 · Pixel Art Maker

一个纯前端、零依赖的网页应用：上传任意图片，一键转换成复古像素风。所有图片处理都在浏览器本地完成，**图片不会上传到任何服务器**。

## 使用

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

## 功能

- 上传方式：点击选择 / 拖拽 / `Ctrl+V` 粘贴剪贴板图片
- 像素块大小调节（马赛克粗细）
- 多种复古调色板：Game Boy / NES / PICO-8 / CGA / 黑白，以及原图色与智能减色
- 智能减色（median-cut 算法自动提取主色，2–64 色）
- Floyd–Steinberg 抖动
- 饱和度 / 对比度微调
- 像素网格线叠加
- 输出放大倍率（1–12×）
- 像素画 / 原图对比，一键下载 PNG

## 技术实现

缩小到低分辨率 → 调色处理（量化 / 抖动）→ 用最近邻插值放大，得到清晰的像素块。纯 HTML + CSS + 原生 JavaScript（Canvas API）。
